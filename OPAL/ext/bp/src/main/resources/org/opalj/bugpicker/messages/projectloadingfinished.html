Running the analysis

Your project has been loaded. You can find some statistics, such as number of classes/methods/etc. by navigating in the menu to File > Project info or pressing Ctrl+I (Cmd+I on MacOS).

You are now ready to run the dead code analysis on your project. To do that, either navigate in the menu to Analysis > Run or press Ctrl+R (Cmd+R on MacOS).

Tip: You can cancel the analysis at any time by pressing the Cancel button in the progress dialog.

Tip: You can view this message again by navigating in the menu to Help > Browse Help > Running the analysis.
